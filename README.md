# CompiladorFrontend

Proyecto frontend en HTML, CSS y JavaScript para la interfaz `NEBULA IDE`.

## Vista previa local

Abre `index.html` en tu navegador.

## Estructura

- `index.html`: punto de entrada de la interfaz.
- `assets/css/styles.css`: estilos visuales de la aplicacion.
- `assets/js/app.js`: logica de la interfaz, compilacion, historial y conexion con el backend.
- `docs/`: carpeta reservada para documentacion futura del proyecto.
