Reglas de NEBULA

1. Estructura básica

Todo programa debe empezar con Algoritmo Nombre y terminar con FinAlgoritmo.

Algoritmo HolaMundo
Escribir "Hola mundo"
FinAlgoritmo

2. Declarar variables

Usa Definir nombre Como Tipo. Los tipos válidos son Entero, Real, Cadena y Logico.

Definir edad Como Entero
Definir nombre Como Cadena

3. Asignar valores

La asignación se hace con <-.

edad <- 19
nombre <- "Ana"
activo <- Verdadero

4. Mostrar información

Usa Escribir para imprimir texto, variables o expresiones.

Escribir "Hola mundo"
Escribir edad

5. Condicionales

La forma correcta es Si ... Entonces, opcionalmente Sino, y siempre FinSi.

Si edad >= 18 Entonces
Escribir "Mayor"
Sino
Escribir "Menor"
FinSi

8. Comparaciones validas

Los operadores disponibles son ==, !=, <, >, <= y >=.

Si x == 10 Entonces
Escribir "Igual"
FinSi

9. Valores booleanos

Si necesitas valores lógicos, usa exactamente Verdadero y Falso.

Definir activo Como Lógico
activo <- Verdadero

6. Ciclo Mientras

Se abre con Mientras condicion Hacer y se cierra con FinMientras.

Mientras i < 5 Hacer
Escribir i
i <- i + 1
FinMientras

7. Ciclo Para

La estructura soportada es Para variable <- inicio Hasta fin Hacer.

Para i <- 1 Hasta 5 Hacer
Escribir i
FinPara

Hacer / Mientras

También existe una forma tipo repetir-hasta usando Hacer y luego Mientras condicion.

Hacer
Escribir x
x <- x + 1
Mientras x < 5

Funciones

Se declaran con Funcion Nombre(parametros) Como Tipo y se cierran con FinFuncion.

Funcion Sumar(a Como Entero, b Como Entero) Como Entero
Retornar a
FinFuncion

Retornar

Dentro de una función puedes devolver un valor con Retornar.

Retornar resultado

Reglas útiles

Los nombres deben empezar con letra. No puedes usar una variable si no la declaraste antes.

Definir x Como Entero
x <- 10
